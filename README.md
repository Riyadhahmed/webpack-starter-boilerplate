# Webpack Starter boilerplate   
Simple webpack boilerplate with bootstrap, webpack html plugin, file loader, Webpack Dev Server etc   

Installation :    
Clone the projects   
   
   git clone   

 Run the project    

 npm start for develpoment mode   

 npm run build for production
